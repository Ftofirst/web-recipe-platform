# web-recipe-platform
ini bakal jadi web untuk menyebar luaskan sebuah resep. tapi sebenarnya yang tujuan utama pembuatan repo dan projek ini adalah untuk belajar api endpoint php native, konsep-konsep dasar web dan banyaknya teknik ml atau ds yang di implementasikan sebagai fitur untuk admin

## bentuk kasar web recipe platform
akan ada banyak fitur dasar yang akan menjadi mvp alpha untuk standard pembuatan web dasar dan semuanya harus digunakan atau di buat dengan php, js, html, css secara native atau tanpa framework tujuannya adalah untuk meningkatkan pemahaman dasar sebelum beralih ke teknologi yang lebih tinggi.

### mvp alpha
mvp alpha bukan sebuah tonjolan atau highlight dari web ini tapi sebuah fitur yang harus dimiliki untuk memenuhi standard minimal
1. login, cek_login, ragister, insert_register, index.php
2. index.php menampilkan card yang dirender dengan js di card itu harus sudah terhubung dengan database
3. routing ke halaman lain
4. pages_routing = home, profil, upload_resep, dashboard
5. like, dislike, subscribe, report
6. komen
